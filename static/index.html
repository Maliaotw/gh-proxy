🚀 檔案加速下載

💡 使用說明

• 專為解決檔案下載速度慢而設計的加速工具

• 支援多種檔案類型：原始碼、發布版本、單個檔案等

• 只需複製貼上檔案連結，即可獲得加速下載

• ⚠️ 限制：不支援整個專案資料夾的批量下載

📝 輸入範例

GitHub 分支原始碼：https://github.com/username/project/archive/master.zip

GitHub release 原始碼：https://github.com/username/project/archive/v0.1.0.tar.gz

GitHub release 檔案：https://github.com/username/project/releases/download/v0.1.0/example.zip

GitHub 分支檔案：https://github.com/username/project/blob/master/filename

其他網站檔案：https://example.com/path/to/file.zip
